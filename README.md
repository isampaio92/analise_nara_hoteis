
# Análise Nara Hoteis 2025

Projeto tem como objetivo a tomada de decisão de negócio no ramo hoteleiro. Verificando as principais métricas estatísticas.


## Tecnologias

**Análise:** Python, Pandas, NumPy, Matplotlib, Power BI

**Banco de dados:** MySQL


## Como executar

Instalação usando pip install

```bash
  pip install -r requirements.txt
```
